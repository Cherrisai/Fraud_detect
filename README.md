# Fraud Analytics Dashboard

An interactive fraud detection dashboard built using Streamlit.

This project analyzes fraudulent transactions and provides KPI insights, fraud trends, anomaly detection, and predictive analytics.

## Features

• KPI Dashboard for fraud statistics  
• City wise fraud analysis  
• Age wise fraud analysis  
• Time based fraud trend analysis  
• Future fraud prediction  
• AI anomaly detection using Isolation Forest  
• Live fraud simulation  
• Download filtered data option  

## Dataset

fraud_dataset.csv

Columns used:

Time  
City  
Amount  
Fraud  
Age  

## Technologies Used

Python  
Pandas  
NumPy  
Plotly  
Scikit-learn  
Streamlit  

## How to Run

Install dependencies

pip install -r requirements.txt

Run the dashboard

streamlit run app.py

## Output

Interactive fraud analytics dashboard showing fraud patterns across city, age, and time.
